## postcss

+ 用 js 实现的 css 的抽象语法树**AST**(`Abstract Syntax Tree`)

+ `postcss` 只是一个工具，本身不会对 css 一顿操作，它通过插件实现功能，`autoprefixer` 和 `cssNext` 就是其一
