# Vector3 拷贝 与 克隆

## 拷贝 .copy ( v : Vector3 ) : this

+ 通过向量复制方法 `.copy()`，把A和B点的坐标赋值给两个表示网格模型对象,可视化A点和B点

  ```js
  // 两个小球网格模型可视化A点和B点
  mesh1.position.copy(A);
  mesh2.position.copy(B);
  ```

## 克隆 .clone () : Vector3

+ 返回一个新的 `Vector3` ，其具有和当前这个向量相同的x、y和z

  ```js
  const A = new THREE.Vector3(30, 0, 0);
  const B = A.clone();
  ```
