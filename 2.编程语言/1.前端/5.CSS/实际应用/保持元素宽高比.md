# 保持元素宽高比

## 方法1

+ `aspect-ratio: 宽 / 高;`
+ 存在兼容性

  ```css
  .item {
    width: 50%;
    /* height: ?; */

    /* 4/3 宽与高的比例 */
    aspect-ratio: 4 / 3;
  }
  ```

## 方式2

+ 实现

  ```html
  <style>
    .item {
      widht: 50%;
      margin: 0 auto;
    }

    .inner {
      widht: 100%;
      height: 0;
      padding-bottom: 75%;

    }
  </style>

  <div class="item">
    <div class="inner"></div>
  </div>
  ```
