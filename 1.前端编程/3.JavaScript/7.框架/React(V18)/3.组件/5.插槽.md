# 插槽

## 概述

+ 通过 props.children，可以实现类似于 Vue 中插槽的功能，例如：

  ```js
  class 组件B extends React.Component{
    constructor(props){
      super(props);
    }
    render(){
      return (
        <div>
            {this.props.children}
        </div>
      );
    }
  }
  class 组件A extends React.Component{
    constructor(props){
      super(props);
    }
    render(){
      return (
        <组件B>
          <p>Hello, React</p>
          <p>Hello, Redux</p>
          <p>Hello, Facebook</p>
          <p>Hello, Google</p>
        </组件B>
      );
    }
  }
  ```
