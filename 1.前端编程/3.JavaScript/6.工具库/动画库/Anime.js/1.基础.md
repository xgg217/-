# 基础

## 下载

+ npm

  ```bash
  npm install animejs --save

  ts 类型
  npm i --save-dev @types/animejs
  ```

+ 用法

  ```js
  // ES6 模块
  import anime from 'animejs/lib/anime.es.js';

  // CommonJS
  const anime = require('animejs');
  ```

## 示例

+ code

  ```js
  anime({
    targets: 'div',
    translateX: 250,
    rotate: '1turn',
    backgroundColor: '#FFF',
    duration: 800
  });
  ```
