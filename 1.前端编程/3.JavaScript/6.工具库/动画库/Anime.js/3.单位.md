# 单位

## 无单位赋值

+ 对于 `px` 之类的单位可以省略

  ```js
  anime({
    targets: '.unitless-values-demo .el',
    translateX: 250, // -> '250px'
    rotate: 540 // -> '540deg'
  });
  ```

## 有单位数值

+ 强制动画使用某个单位并自动转换初始目标值

  ```js
  anime({
  targets: '.specific-unit-values-demo .el',
    width: '100%', // -> from '28px' to '100%',
    easing: 'easeInOutQuad',
    direction: 'alternate',
    loop: true
  });
  ```

## 相对数值

+ 添加，减去或乘以原始值
+ 有 `+=` 、 `-=` 、 `*=` 这三个值

  ```js
  // 宽度变为原来的两倍，高度度减少50px
  anime({
    targets: '.item',
    width: {
      value: '*=2',
      duration: 1800,
      easing: 'easeInOutSine'
    },
    height: {
      value: '-=50px',
      duration: 1000,
      delay: 1800,
      easing: 'easeInOutSine'
    }
  });
  ```
