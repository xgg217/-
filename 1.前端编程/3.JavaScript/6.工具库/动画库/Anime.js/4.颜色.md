# 颜色

## 概述

+ anime.js 接受并转换Haxadecimal(十六进制)，RGB，RGBA，HSL和HSLA颜色值
+ 注：不支持颜色名称，比如red

  ```js
  .item {
    position: relative;
    width: 100px;
    height: 100px;
    margin-bottom: 10px;
    background-color: #61afff;
  }

  anime.timeline({
    endDelay: 1000,
    easing: 'easeInOutQuad',
    direction: 'alternate',
    loop: true
  })
    .add({ targets: '.item', background: '#555851' }, 0)
    .add({ targets: '.item', translateX: 270 }, 0);
  ```
