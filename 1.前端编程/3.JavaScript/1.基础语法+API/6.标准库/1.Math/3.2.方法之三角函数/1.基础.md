# 基础

## 概述

+ 三角函数 `sin()`、`cos()`、`tan()`、`asin()`、`acos()`、`atan()` 和 `atan2()` 返回的值是*弧度*而非角度

## 角度 --> 弧度

+ 弧度 = 角度 * 2π / 360 = 角度 * π / 180 = 角度 * Math.PI / 180

  ```js
  // 90°转为弧度
  cons rad = 90 * Math.PI / 180 // 1.5707963267948966
  ```

## 弧度 --> 角度

+ 角度 = 弧度 *360 / 2π = 弧度 *180 / π = 弧度 * 180 / Math.PI

  ```js
  // 1.57 弧度转角度
  cons deg = 1.57 * 180 / Math.PI // 90
  ```
