# 全局引入与 局部引入

## 全局引入

+ 异步组件全局引入

  ```js
  app.component(
    'MyComponent',
    defineAsyncComponent(() => import('./components/MyComponent.vue'))
  )
  ```

## 局部引入

+ 局部引入

  ```html
  <script>
    import { defineAsyncComponent } from 'vue'

    export default {
      components: {
        AdminPage: defineAsyncComponent(() => import('./components/AdminPageComponent.vue')),
      },
    }

  </script>

  <template>
    <AdminPage />
  </template>
  ```
