# 类修饰符

## public

+ `@public` 标识类的属性或方法的访问范围是public

  ```js
  /**
   * @class
   */
  function myClass() {
      ...
  }

  /**
   * @method
   * @public
   * @param {string} a - 参数a
   */
  myClass.prototype.foo = function(a) {
      ...
  }
  ```

  /**

## protected

      ...
  }

  /**

## instance

+ `@instance` 实例属性或方法的标识

  ```js
  /**
   * @class
   */
  function myClass() {
      ...
      /**
       * @member {string}
       * @instance
       */
      this.foo = 'a';
  }
  ```

## static

+ `@static` 静态属性或方法的标识

  ```js
  /**
   * @class
   */
  function myClass() {
      ...
  }

  /**
   * @static
   */
  myClass.foo = 'a';
  ```
