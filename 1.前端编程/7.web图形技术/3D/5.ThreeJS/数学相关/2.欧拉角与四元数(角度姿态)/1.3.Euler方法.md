# Euler 方法

## .copy ( euler : Euler ) : this

+ 将 `Euler` 的属性拷贝到当前对象

  ```js
  const Euler = new THREE.Euler();
  Euler.x = Math.PI / 3;
  Euler.y = Math.PI / 3;

  // 属性拷贝到 fly
  fly.rotation.copy(Euler);
  ```

## .clone () : Euler

+ 返回一个与当前参数相同的新欧拉角

## .equals ( euler : Euler ) : Boolean

+ 检查 `euler` 是否与当前对象相同

## .set ( x : Float, y : Float, z : Float, order : String ) : this

+ 设置该欧拉变换的角度和旋转顺序 order。

  + `x` - 用弧度表示x轴旋转量
  + `y` - 用弧度表示y轴旋转量
  + `z` - 用弧度表示z轴旋转量
  + `order` - (optional) 表示旋转顺序的字符串
