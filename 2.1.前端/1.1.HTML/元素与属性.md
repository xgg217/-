# 元素与属性

## 注意

+ 区分大小写

## 元素

+ 代码

  ```html
  <a href="https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png"></a>
  ```

## 空元素

+ 没有结束标记

  ```html
  <img src="" alt="">

  <!-- 也可以写成 -->
  <img src="" alt="" />
  ```

## 元素组成部分

+ 元素 = 起始标记 + 结束标记 + 元素内容 + 元素属性

## 属性

+ 属性 = 属性名 + 属性值

## 属性分类

+ 局部属性:某些元素特有属性

+ 全局属性:所有元素通用
