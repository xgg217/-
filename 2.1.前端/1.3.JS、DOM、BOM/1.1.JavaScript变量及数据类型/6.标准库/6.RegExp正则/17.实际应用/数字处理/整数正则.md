# 整数

## 正整数正则

+ 正整数正则

  ```js
  var posPattern = /^\d+$/;

  // 输出 true
  console.log(posPattern.test("42"));
  ```

  ```js
  Number.isInteger(Number(value))

  Number.isInteger(Number('')) // true
  Number.isInteger(Number('false')) // false
  Number.isInteger(Number('12')) // true
  Number.isInteger(Number(0)) // true
  ```
