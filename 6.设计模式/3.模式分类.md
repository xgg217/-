# 模式分类

## 概述

+ 在这 23 种模式中，可以根据意图或者目的来进行分类：

  + 创建型模式：提供一种**创建对象的机制**，增加已有代码的灵活性和复用性
  + 结构型模式：介绍如何**将对象和类组装成较大的结构**，并同时保持结构的灵活和高效
  + 行为型模式：负责**对象间的高效沟通和职责分配**
