# 类型转换

## int(x)

+ `int(x)`将 `x` 转换为整数（直接舍去小数部分），`x` 可以是浮点数或字符串

  ```py
  int(4.4) # 4

  int(4.9) # 4
  ```

+ 布尔值 `True` 转为1，`False` 转为 0

  ```py
  int(True) # 1
  int(False) # 0
  ```

+ 字符串符合整数格式（浮点数格式不行）则直接转成对应整数，否则报错

  ```py
  int('4') # 4

  int('4.9') # 程序报错
  int('12a') # 程序报错
  int('a12') # 程序报错
  ```

+ 改变进制

  ```py
  int('22', 16)
  ```

## 自动转型

+ 整数和浮点数混合运算是，表达式结果自动转型为浮点数

  ```py
  2 + 8.0 # 10.0
  ```
