# =赋值运算符

## 使用

+ 赋值运算符（Assignment Operators）用于给变量赋值

+ 最常见的赋值运算符，当然就是等号（`=`），表达式 `x = y` 表示将y的值赋给x

+ 除此之外，JavaScript还提供其他11个复合的赋值运算符

  ```js
  x += y // 等同于 x = x + y
  x -= y // 等同于 x = x - y
  x *= y // 等同于 x = x * y
  x /= y // 等同于 x = x / y
  x %= y // 等同于 x = x % y
  x >>= y // 等同于 x = x >> y
  x <<= y // 等同于 x = x << y
  x >>>= y // 等同于 x = x >>> y
  x &= y // 等同于 x = x & y
  x |= y // 等同于 x = x | y
  x ^= y // 等同于 x = x ^ y

  ```

+ 这些复合的赋值运算符，都是先进行指定运算，然后将得到值返回给左边的变量
