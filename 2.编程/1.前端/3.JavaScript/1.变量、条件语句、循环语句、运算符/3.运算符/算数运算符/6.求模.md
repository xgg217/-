# 求模

## 原理

+ `x 求模 y`：表示 `x - (n * y)`

+ `n` 表示商的取值 `（n = x / y）`，直接去掉 `n` 的小数，向下取整

  ```js
  7 求模 3 = 1

  // 表示
  7 / 3 = 2.333 = 2
  7 - (2 * 3) = 1
  ```

  ```js
  7 求模 -3 = -2

  // 表示
  7 / 3 = 2.333 = -3
  7 - (-3 * -3) = -2
  ```

  ```js
  -7 求模 3 = 2

  // 表示
  -7 / 3 = 2.333 = -3
  -7 - (-3 * 3) = 2
  ```
