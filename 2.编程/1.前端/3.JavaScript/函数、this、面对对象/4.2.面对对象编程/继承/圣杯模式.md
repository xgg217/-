# 圣杯模式

## 概述

+ 圣杯模式

  ```js
  /**js继承(圣杯模式)
  * Target继承Origin
  * @param {*} Target
  * @param {*} Origin 被继承
  */
  var inherit = (function (){
      var F = function () {};
      return function(Target, Origin) {
          F.prototype = Origin.prototype;
          Target.prototype = new F();
          Target.prototype.constructor = Target;
          Target.prototype.uber = Origin.prototype; // 标记继承自谁
      }
  }());
  ```
