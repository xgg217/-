# toFixed

## 作用

+ `toFixed()` 方法可把 Number 四舍五入为指定小数位数的数字

  ```js
  (10).toFixed(2) // "10.00"
  10.005.toFixed(2) // "10.01"
  ```

## 注意

+ 四舍五入的规则与数学中的规则不同，使用的是银行家舍入规则
+ 银行家舍入规则：四舍六入五考虑，五后非零就进一，五后为零看奇偶，五前为偶应舍去，五前为奇要进一

  ```js
  (10.055).toFixed(2) // '10.05'
  (10.005).toFixed(2) // '10.01'
  ```
