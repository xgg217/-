# 列表元素

## 有序列表

+ `ol`

+ `li`

  ```html
  <ol>
    <li>士大夫</li>
    <li>士大夫</li>
    <li>士大夫</li>
  </ol>
  ```

## 无序列表

+ `ul`

+ `li`

  ```html
  <ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
  </ul>
  ```

## 定义列表

+ 通常用于一些术语的定义

+ `dl`

+ `dt`

+ `dd`

  ```html
  <dl>
    <dt>html</dt>
    <dd>超文本标记语言</dd>

    <dt>元素</dt>
    <dd>组成html单元格</dd>
  </dl>
  ```
