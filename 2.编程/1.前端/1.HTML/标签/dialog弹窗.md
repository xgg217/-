# dialog弹窗

## 概述

+ `<dialog>` 元素表示一个对话框或其他交互式组件，例如一个可关闭警告、检查器或者窗口

  ```html
  <dialog open>
    <form method="dialog">
      <p>Do you want to confirm your action?</p>
      <div class="right">
        <input class="btn" type="submit" value="Ok" />
        <input class ="btn" type="submit" value="Cancel" />
      </div>
    </form>
  </dialog>
  ```
