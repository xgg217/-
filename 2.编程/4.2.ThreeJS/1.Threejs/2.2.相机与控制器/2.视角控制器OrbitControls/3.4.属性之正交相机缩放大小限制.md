# 属性之正交相机缩放大小限制

## 属性

+ `.maxZoom` : Float

  + 你可以将相机放大多少，该属性仅对正交相机 `OrthographicCamera` 生效
  + 默认值 Infinity


+ `.minZoom` : Float

  + 可以将相机放大多少，该属性仅对正交相机 `OrthographicCamera` 生效
  + 默认值

## 正投影缩放范围

+ 对于正投影相机对象OrthographicCamera，可以通过相机控件的 `.minZoom` 和 `.maxZoom` 属性实现

  ```js
  // 缩放范围
  controls.minZoom = 0.5;
  controls.maxZoom = 2;
  ```
