# 进制转换

## 方法

1. 换基法(换向十进制)
2. 除余法（十进制向其他进制转换）
3. 按位拆分法
4. 按位合并法

  ![进制转换](./images/进制转换.png)

## 换基法

+ 给出数值转换的定量方法，对于任意一个基数 N 进制下的数字 x 转换为十进制的方法

  ![换基法](./images/换基法.png)

+ 示例

  ```js
  // 十进制 2020
  2020=X=X4 X3 X2 X1
  2020= 2 * 10³ + 0 * 10² + 2 * 10 + 0 * 10º
  ```

  ![十进制的2020](./images/十进制的2020.png)
