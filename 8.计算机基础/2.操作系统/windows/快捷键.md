# 快捷键

## 虚拟桌面

+ 创建虚拟桌面

  ```
  window + ctrl + D
  ```

+ 切换

  ```
  window + ctrl + 左右方向键
  ```

+ 关闭虚拟桌面

  ```
  window + ctrl + F4
  ```

## 从程序到桌面

+ 回到桌面

  ```
  window + M
  ```

+ 查看桌面

  ```
  window + ,
  ```

## 任务栏打开程序

+ 任务栏打开程序

  ```
  window + 1~9
  ```

## 打开的应用程序列表

+ 查看

  ```
  ctrl + alt + tab
  ```

+ 任务视窗查看

  ```
  window + tab
  ```

+ 快速切换程序

  ```
  alt + tab
  ```

## 打开任务管理器

+ 打开任务管理器，同时退出无响应的程序

  ```
  ctrl + shift + esc
  ```

## 打开文件夹

+ 快速打开

  ```
  window + E
  ```

## 使用放大镜

+ 放大镜

  ```
  window + +/-
  ```

## 打开任务操作中心

+ 任务操作中心s

  ```
  window + A
  ```

## 打开设置

+ 打开设置

  ```
  window + I
  ```
