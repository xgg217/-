# 变量名规范

## 要求

1. 变量名必须只包含：字母、数字、下划线

    ```go
    var %1 string // 错误
    var $ string // 错误
    ```

2. 首个字符不能为数字

    ```go
    var 1 string // 错误
    ```

3. 不能使用Go内置的25个关键字

## 命名建议

+ 变量的命名规则遵循骆驼命名法，即首个单词小写，每个新单词的首字母大写，例如：`numShips` 和 `startDate`

+ 见名知意，通过变量名就可以知道变量所代指的值，例如：`name` 、 `email`
