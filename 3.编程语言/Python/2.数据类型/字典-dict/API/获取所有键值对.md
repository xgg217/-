# 获取所有

## items() 获取所有键值对

+ 通过 `items()` 获取所有键值对

  ```py
  a = {'name': '字符串', 'age': 18}
  a.items() # dict_items([('name', '字符串'), ('age', 18)])
  ```

## keys() 获取所有键

+ `keys()` 获取所有键

  ```py
  a = {'name': '字符串', 'age': 18}
  a.keys() # dict_keys(['name', 'age'])
  ```

## values() 获取所有值

+ `values()` 获取所有值

  ```py
  a = {'name': '字符串', 'age': 18}
  a.values() # dict_values(['字符串', 18])
  ```
