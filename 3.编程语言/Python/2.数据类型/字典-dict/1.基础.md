# 字典

## 特点

+ 无序可变序列
+ 使用键-值（`key-value`）存储，具有极快的查找速度
+ 通过 `key` 实现快速获取，删除、更新对应的 `value`
+ 一个 `key` 只能对应一个 `value`，所以，多次对一个 `key` 放入 `value` ，后面的值会把前面的值冲掉

  ```py
  d['Jack'] = 90
  d['Jack']
  # 90

  d['Jack'] = 88
  d['Jack']
  # 88
  ```

+ `key` 是任意不可变的数据。如：整数、浮点数、字符串、元组。但是列表、字典、集合这些可变对象不能作为 `key` 。`key` 不能重复

+ `value` 可以是任意的数据，并且可以重复
