# 字符串比较

## 字符串驻留机制

+ 仅保存一份相同且不可变字符串的方法，不同的值被存放在字符串驻留池中
+ Python 支持字符串驻留机制，对于符合标识符的字符串（仅包含下划线_、字符和数字）会启用字符串驻留机制

  ```py
  # 符合驻留规则
  a = "abc_123"
  b = "abc_123"

  a is b # True
  ```

  ```py
  # 不符合驻留规则
  a = "dd#"
  b = "dd#"

  a is b # False 比较地址
  a == b # True 比较值
  ```

  ```py
  str1 = "aa"
  str2 = "bb"

  str1 + str2 == "aabb" # True
  ```

## 重点

+ 使用 `==` `!=` 对字符串进行比较，是否含有相同的字符
+ 使用 `is` `not is` ，判断两个对象是否是同一个对象。比较的是对象的地址，即 `id(obj1)` 与 `id(obj2)` 是否相等
