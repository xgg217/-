# 运算函数

## abs(x)

+ x 的绝对值

## divmod(x,y)

+ 求整和求余
+ (x//y, x%y),输出为二元组形式（也称为元组类型）

  ```py
  divmod(10,3)
  # (3, 1)
  ```

## pow(x,y[,z])

+ (x**y)%z
+ pow(x, y) 与 x**y 相同

## round(x[,ndigits])

+ 对 x 四舍五入，保留ndigits 位小数
+ round(x) 返回四舍五入的整数值

## max(x1, x2,...xn)

+ x1, x2,...xn 的最大值，n 没有限定

## min(x1, x2,...xn)

+ x1, x2,...xn 的最小值，n 没有限定
