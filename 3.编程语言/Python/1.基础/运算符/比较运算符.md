# 比较运算符

## 概述

+ 所有比较运算符返回 1 表示 True，返回 0 表示 False

+ 一下假设 a = 15， b = 30

|运算符|描述                              |示例|
|:-    |:-                               |:-                   |
|==    | 等于。比较对象的值是否相等        | (a == b) 返回 False |
|!=    | 不等于。比较两个对象的值是否不相等 | (a != b) 返回 True |
|>     | 大于                             | (a > b) 返回 False |
|<     | 小于                             | (a < b) 返回 True  |
|>=    | 大于等于                         | (a >= b) 返回 False|
|<=    | 小于等于                         | (a <= b) 返回 True |
