# 基础

## API

+ len(a) 字符串长度
+ a.startswith("我是小刚刚") 指定字符串开头
+ a.endswith("我是小刚刚") 指定字符串结尾
+ a.find("小刚刚") 第一次出现的位置
+ a.rfind("小刚刚") 最后一次出现的位置
+ a.count("小刚刚") 指定字符串出现的次数
+ a.isalnum() 所有字符串全是字母或数字
