# 向量运算

## 零向量

+ 零向量：长度等于0的向量（向量里面是有的值都是0）叫做零向量，记作0
+ 没有方法的向量

  ```
  [0,0,0]
  ```

## 标准化向量

+ 单位向量：模等于1个单位长度的向量叫做单位向量
+ 作用：对长度不关心，只关心方向
