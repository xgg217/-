# 任意角的度数

## 概念

+ 任意角：一条射线绕着它的端点在平面内旋转形成的图形

  ![任意角](./images/任意角.png)

## 分类

+ 正角：按*逆时针*方向旋转形成的角
+ 负角：按*顺时针*方向旋转形成的角
+ 零角：一条射线没有做任何旋转形成的角（零角的始边与终边重合）

  ![任意角度数示例](./images/任意角度数示例.jpg)

+ 注意

  + 角度的范围不在受限于 0° ~ 360°
  + 确定任意角的度数要抓住旋转方向及旋转圈数
  + 引入正、负角的概念后，角的加减运算类似于实数的加减运算

  ```js
  // 130° 是以 50° 角的终边为始边，逆时针旋转 80° 形成的角
  50° + 80° = 130°

  // -30° 是以 50° 角的终边为始边，顺时针旋转 80° 形成的角
  50° - 80° = -30°
  ```
